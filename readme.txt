== Description ==
Are you working on new Gutenberg editor and want to add map on your page? Please use this block to add Google Map. Install, Activate and Add your address. It is simple to use and user friendly. You just need to enter address and Google Map API key. 

Are you still working on classic editor? WordPress has recently launched new editor called Gutenberg. Discover more about the project. (<a href="https://wordpress.org/gutenberg/" target="_blank">https://wordpress.org/gutenberg/</a>).


= Plugin Features =

It works like all other Gutenberg block. Add following things and it should work smoothly. 

* Address
* Adjust Map height as needed
* API key (How to create API key) <a href="https://developers.google.com/maps/documentation/javascript/get-api-key" target="_blank">https://developers.google.com/maps/documentation/javascript/get-api-key</a>


== Installation ==

Standard stuff. Make sure you have the Gutenberg plugin installed, obviously.


== Frequently Asked Questions ==

**Map is not working?**
Please make sure you configure and add Map API key correctly. Map will not work if API key is wrong. 

**How to create API key?**
API key (how to create API key) <a href="https://developers.google.com/maps/documentation/javascript/get-api-key"  target="_blank">https://developers.google.com/maps/documentation/javascript/get-api-key</a>
